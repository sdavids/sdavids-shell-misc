= AsciiDoc

This section contains scripts related to https://docs.asciidoctor.org/asciidoc/latest/[AsciiDoc]:

xref:scripts/asciidoc/antora-build.adoc[]:: create a documentation site with Antora from a given directory
xref:scripts/asciidoc/asciidoc-html-build.adoc[]:: typeset the documents of a given directory into HTML
xref:scripts/asciidoc/asciidoc-pdf-build.adoc[]:: typeset the documents of a given directory into PDFs

[NOTE]
====
This documentation site is built with Antora and hosted on https://docs.github.com/en/pages/quickstart[GitHub Pages].

The sources are available at https://github.com/sdavids/sdavids-shell-misc/blob/main/docs/[docs].

The Antora playbook can be accessed at https://github.com/sdavids/sdavids-shell-misc/blob/main/docs/antora-playbook.yml[antora-playbook.yaml].

The GitHub Action for publishing this documentation site is available at https://github.com/sdavids/sdavids-shell-misc/blob/main/.github/workflows/publish.yml[publish.yml].
====
