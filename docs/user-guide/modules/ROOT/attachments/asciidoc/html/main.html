class Main {
  // tag::main[] (1)
  void main() {
    System.out.println("Hello, World!");
  }
  // end::main[] (2)
}
1 // tag::main[] starts a tag named "main"
2 // end::main[] ends the tag

Just the method defined with the main the tag and fix indentation:

void main() {
  System.out.println("Hello, World!");
}
