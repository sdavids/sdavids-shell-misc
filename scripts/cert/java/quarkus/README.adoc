== With standalone certificates

[,console]
----
$ ../../create_self_signed_cert.sh
$ ./gradlew quarkusDev
----

=> https://localhost:3000

== With certificate authority based certificates

[,console]
----
$ ../../create_ca.sh
$ ../../create_ca_based_cert.sh
$ ./gradlew quarkusDev
----

=> https://localhost:3000
